
Project Overview

This project contains an automated script that opens the Amazon shopping website and checks whether the Rufus icon is visible or not. is displayed on the page.


Features

Checks for the presence of the Rufus icon on the webpage.

Provides a simple true/false or pass/fail result based on the icon's visibility.
